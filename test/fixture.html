# プロダクト仕様・設計案

## 1. プロダクト概要
* **コンセプト**: 「教え方(教員・授業)を変えるのではなく、教わり方(生徒・受講時の認知)を変える」
* **目的**: 生徒が自発的に「問いの生成」「知識の接続」を行える状態を作る
* **構成**: ソフトウェア(AI)とハードウェア(物理デバイス)のハイブリッド

---

## 2. システム構成と役割

### ① ハードウェア(物理デバイス)
* **役割**: スマートフォン等の誘惑から生徒を切り離す
* **主な機能要件**:
    * 持ち運び可能かつシンプルなUI
    * 物理ボタンによるタイムスタンプ記録
    * Bluetooth等によるデータ同期

### ② ソフトウェア(AI・アプリケーション)
| 機能 | 説明 |
| --- | --- |
| 事前仕込み | 教科・単元に応じた「問いのプロンプト」を提示 |
| 事後振り返り | AI対話で「なぜあの瞬間気になったのか」を深掘り |
| 可視化 | 生成した問いをマップ化し探究力を定量化 |

> 「教わり方」を変えるというコンセプトが core value。
